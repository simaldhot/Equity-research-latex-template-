# Equity-research-latex-template-
Equity Research LaTeX Template
This folder contains theLaTeX template for writing equity research reports.
For almost all users, you only need to edit main.tex. This is where all writing and content should be done.
Which files to edit
main.tex
This is the main file. Edit the title, authors, sections, text, tables, equations, and citations here.

Which files NOT to edit
oup-contemporary.cls
Do not edit this file, except for one small change described below. Editing anything else in this file may break the formatting.
One required change in the class file
There is one line in oup-contemporary.cls that must be updated for each report.
Open oup-contemporary.cls and search for:
Current company Price
Change this line to match your company and price
Do not change anything else in the class file.
